# Loan Ledger — EMI Worksheet

A simple, free EMI (Equated Monthly Installment) calculator. Enter a loan amount, interest rate, and tenure to see the real monthly payment, the principal-vs-interest split, and a full year-by-year amortization breakdown — not just a single number.

**Live demo:** https://emi-calculator-beta-flax.vercel.app/

## Why I built this

Most EMI calculators online only show the monthly payment. This one breaks down how much of every rupee paid actually goes toward interest vs. the loan itself, and how that changes year by year as the loan gets paid down.

## Features

- Real-time EMI calculation as you move the sliders
- Principal vs. interest visual split
- Total interest payable and total amount payable
- Full year-by-year amortization ledger (principal paid, interest paid, balance remaining)

## Tech stack

Plain HTML, CSS, and JavaScript — no frameworks, no build step. Deploys as a static site.

## Run locally

Clone the repo and open `index.html` directly in any browser:

```bash
git clone https://github.com/Baala-Murugan-K/EMI-CALCULATOR.git
cd EMI-CALCULATOR
open index.html
```
